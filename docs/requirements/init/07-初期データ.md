## 14. 実行手順

```bash
# 1. マイグレーションを順次適用（04-DB設計.md のファイル群）
npx supabase db push

# 2. seed.sql を実行
npx supabase db reset --seed   # ローカル開発用
# OR 本番: SQL エディタから手動実行

# 3. 動作確認
# - /items に Item 一覧が出る
# - /items/:id を開いて見積もりパネルが動く
# - 設定画面でマスタが見える
```

## 15. 最小デモシナリオ

seed 後、以下が動けば成功:

1. `/items/ABC-001` を開く
2. 材料タブ → [+ 追加] → 「板金材 → SS400」を選ぶ
3. 葉に到達して計算式が確定（パス変数で比重・単価が自動セット）
4. 直径 50, 長さ 200 を入力
5. 小計が `(50/2)² × π × 200 × 7.85/1000000 × 380 × 1.10 ≈ 1284 円` で出る
6. 工程タブ → 旋盤加工 → 加工時間 0.5h → 小計 3000 円
7. 中計（材料）= 1284、中計（工程）= 3000
8. 合計 = (1284 × 1.10) + (3000 × 1.30) = 1412 + 3900 = 5312 → 100 円丸めで 5300 円

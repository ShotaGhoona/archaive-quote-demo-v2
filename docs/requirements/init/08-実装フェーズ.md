## トラブルシューティング Tips

### 計算が合わない

1. 計算式の変数コードが `variable_definitions` と一致しているか確認
2. PATH 変数の `path_key` が経路上のノード値 JSON のキーと一致しているか
3. LOOKUP の軸値の型（文字列 vs 数値）に注意

### 上書き 3 レバーの計算

`adjust(x) = override ?? (x * multiplier + adjustment)` の式はサーバー側 1 箇所に集約。フロントは値を渡すだけ。

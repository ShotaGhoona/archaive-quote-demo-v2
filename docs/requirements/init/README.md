# Archaive-Quote v2 要件定義

製造業向け汎用原価見積モジュール **Archaive-Quote** の **ビジネス要件・DB 構造・UI 仕様** をまとめた資料。

## このフォルダの位置づけ

「何を作るか」に絞った要件定義。技術スタック・実装パターンは扱わない（CLAUDE.md・実装コードを参照）。

実装が進むにつれて要件が変わった箇所も、このフォルダでは **実装後の現状** を反映する。
